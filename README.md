# Gallus Genetics Resource Center — gallusresources.org

Static website for the Gallus Genetics Resource Center, a Connecticut nonprofit
that maintains transgenic and wild-type chicken lines and ships fertilized eggs
to research laboratories. Served via GitHub Pages.

## Lines featured
- **Wildtype Red Star** — available now (foundation flock)
- **Cytoplasmic Green** — coming soon (ubiquitous cytoplasmic GFP)
- **Membrane Green** — coming soon (membrane-targeted GFP)
- **Mitochondria Green** — coming soon (mitochondria-targeted GFP)

## Files
- `index.html` — the single-page site (HTML + CSS + JS inline)
- `404.html` — custom not-found page
- `hero-chicken.png` — standing hen, cresting the top bar in the hero
- `contact-chicken.png` — pecking hen, contact section
- `404-chicken.png` — startled hen, 404 page
- `CNAME` — custom domain (gallusresources.org)

## Design
Black/white with #007FFF blue and orange accents, Space Grotesk + Inter,
hard 2px borders with offset shadows, and a Bayer-dithered backdrop in the
mission section. The hero hen is layered above the nav bar so her comb
appears to break out over it. Contact email: admin@gallusresources.org
